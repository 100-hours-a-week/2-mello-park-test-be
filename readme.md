## 📂 프로젝트 구조

## 🚀 시작하기
### 1️⃣ 레포지토리 클론
```sh
git clone https://github.com/100-hours-a-week/2-mello-park-test-be.git
cd express-study
```
### 2️⃣ 의존성 설치
```sh
npm install
```
### 3️⃣ 서버 실행
```sh
npm start
```
또는 `nodemon`을 사용할 경우
```sh
npm run dev
```
서버 실행 후 `http://localhost:3000`에서 확인
